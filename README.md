# Darwin Bot 🧬🤖

An interactive digital assistant and evolutionary robotics simulation. This project allows users to develop and train virtual creatures using machine learning to master locomotion in a dynamic environment. 

## 🚀 Key Features
* **Evolutionary Algorithms:** Utilizes NEAT (NeuroEvolution of Augmenting Topologies) to train and optimize the neural networks of the bots.
* **Physics-Based Arena:** Creatures learn to navigate and balance in a simulated physical environment.
* **Custom Fitness Functions:** Algorithms designed to reward successful movement, efficient walking gaits, and path planning.
* **Sensor Fusion & Kinematic Control:** Integrates real-time environmental feedback for dynamic motor control.
